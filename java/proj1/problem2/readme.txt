The program to run is called MatmultD.java. To compile it, write the following command:
"$> javac MatmultD.java"
It will create .class files. Now, to run it, write the following command:
"$> java MatmultD [NUM_THREAD=1] < mat5.txt"
You can specify to the program the number of thread you want. By default the number will be 1.
