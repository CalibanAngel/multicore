2 folders can be find, dynamic for Dynamic Load Approach and static for Static Load Approach.
In both of these folder a JAVA file can be find, to compile it, write the following command:
"$> javac myfile.java"
It will create .class files. Now, to run it, write the following command:
"$> java myfile [NUM_THREAD=4]"
You can specify to the program the number of thread you want. By default the number will be 4.
